# termux-dotfiles

minimal termux setup

### includes
- `hushlogin` - hide text on startup
- `nvim` - extremely minimal neovim config  
- `termux` - iosevka bold font

### requirements
- git
- nvim
- stow

### install
```bash
git clone https://github.com/nvimueller/termux-dotfiles.git
cd termux-dotfiles
stow hushlogin nvim termux
